데이터 베이스 -> 로그 스태쉬 -> 엘라스틱 서치 -> 키바나

엘라스틱서치는 jvm 위에서 동작하기 때문에 jdk 8 이상 있어야한다.

엘라스틱서치는 역색인 기능 덕분에 검색의 기능 관점에서는 RDB보다 훨씬 빠르다.

REST API 데이터를 CRUD 조작이 가능하다.
